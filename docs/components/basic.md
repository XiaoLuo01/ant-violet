## Icon - 图标
提供一些常用图标集合。

### 使用方法

<ClientOnly>
  <icon-demo1></icon-demo1>
</ClientOnly>

```HTML
<v-icon name="v-edit"></v-icon>
<v-icon name="v-share"></v-icon>
<v-icon name="v-delete"></v-icon>
<v-button icon="v-search">搜索</v-button>
```

### Attributes
参数 | 说明 | 类型 | 可选值 | 默认值
:-:| :-: | :-: | :-: | :-: 
name | icon 图标名 | String | —— | ——
